# Luxury Watches Design Collection

A visual design project featuring a **22-page luxury watch album** and a **standalone poster**, created as a college assignment and showcased here as a design portfolio piece.

## 📌 Overview

This project explores brand-style editorial layout design for the luxury watch industry. Each spread is built around a single timepiece, pairing product photography with brand identity elements (logos, taglines, and catalog-style specs) to mimic the look of a premium print catalog or lookbook.

## 🎨 Contents

| File | Description |
|---|---|
| `luxury_watches_album.pdf` | 22-page album, one spread per watch (e.g., Rolex Daytona, Patek Philippe Grandmaster Chime), styled as a luxury product catalog |
| `luxury_watch_poster.pdf` | Single-page promotional poster — Rolex Cosmograph Daytona under the tagline *"Success with Time"* |

## 🛠️ Tools & Process

- **Design tool:** Canva
- **Reference & asset research:** Google (product photography, brand typography, layout inspiration)
- **Approach:**
  1. Researched real catalog/poster layouts from luxury watch brands for visual language (typography, negative space, color palette)
  2. Sourced and composited product imagery for each watch
  3. Built a consistent grid/template across all 22 album pages for visual continuity
  4. Designed the poster as a standalone hero piece with atmospheric background and bold typographic focus
  5. Exported all work as print-ready PDFs

## 💡 What I Practiced

- Applying consistent visual branding across a multi-page layout
- Typography hierarchy and composition for premium/luxury aesthetics
- Working within a single design tool (Canva) to produce print-style deliverables
- Translating brand identity cues (logos, taglines, color language) into original layouts
